# Giao diện Quản lý phòng trọ

Frontend React/Vite cho mini project SPQM Level 3.

## Chạy riêng giao diện

```bash
npm install
npm run dev
```

Mở:

```text
http://localhost:5173
```

## Kiểm tra giao diện

```bash
npm run lint
npm run build
```

## Chạy cùng toàn bộ hệ thống

Từ thư mục gốc project:

```bash
docker compose up --build
```

Sau đó mở:

```text
http://localhost:5173
```
